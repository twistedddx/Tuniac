Shortcuts overview

Control shortcut combinations
Shortcut Function
Winkey + Numpad 5 Play/pause
Winkey + Numpad 4 Previous song
Winkey + Ctrl + Numpad 4 Previous song in history
Winkey + Numpad 6 Next song
Winkey + Numpad 8 Volume up
Winkey + Numpad 2 Volume down
Winkey + Numpad 0 Stop
Winkey + Numpad 1 Seek backward
Winkey + Numpad 3 Seek forward
Winkey + Numpad 9 Random next
Ctrl + Spacebar Play/pause
Ctrl + Shift + Spacebar Soft pause
Ctrl + Left arrow Previous song
Ctrl + Clicking the previous button Previous song in history
Ctrl + Right arrow Next song
Ctrl + Shift + Right arrow Random next

Playlist usage shortcuts
Shortcut Function
Alt + Selecting songs Play selected
Delete Delete file/files from playlist and/or media library
Shift + Delete Delete file/files from playlist and/or media library and from the drive
Ctrl + Using the Filter by field function Creates playlist of songs not matching the field
