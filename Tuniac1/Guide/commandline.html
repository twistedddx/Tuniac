Commandline options overview

Commandline player control options
Option Function
-play Play
-pause Pause
-softpause Soft pause
-togglepause Toggle pause
-prev Previous song
-next Next song
-randomnext Random next
-exit Exit
-restore Restore screen
-minimize Minimize screen
-wipeprefs Don't save the preferences on closure
-dontsaveprefs Delete all preferences/file associations from the registry and use default settings
-dontsaveml Don't save the media library on closure
-nofocus Don't take focus
